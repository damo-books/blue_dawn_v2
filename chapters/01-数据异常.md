# 第1章 · 数据异常

高速路的指示牌往天上飘。

纪若寒握着方向盘，揉了第三次眼睛。那牌子还飘着，银灰色铁皮被什么东西揉成一团，像一块被风吹歪的云，不对劲。

导航早废了，屏幕上全是雪花，偶尔蹦出几个乱码：
```
数 据 流 动 中...
编 码 重 构...
```

他瞟了眼副驾的手机，屏保是和苏晴在青岛拍的，她笑的露出虎牙。最后一条短信停在十分钟前：

> 哥，南城不对，所有电子屏幕都在跳，全是奇怪的几何纹...看起来...像星星在流。

纪若寒回：赶紧去研究所找我，我二十分钟到。

发出去，红色感叹号。

然后，整个世界的通讯都断了，像有人一把剪了总网线。

纪若寒挂上空挡，推开车门下来。风里有股铁锈味，混着点电路板烧糊的焦味，闻着不舒服。

抬头看天。

好好的蓝天，裂了一道缝。

缝里漏出淡蓝色的光，像烧化了的蓝宝石，光里有东西在动——仔细看，那动的东西是字：

```
灵界融合启动
99号试炼场锚定
坐标：73.4°E 45.2°N...
```

字在天上流，像LED屏坏了，闪了闪，又变了：

```
叠合进度17%
正在刻印界印
请保持静止...
```

后颈猛地一凉。

好像有什么小东西钻了进去，顺着血管爬，停在左手腕内侧。纪若寒撸起袖子，看见皮肤底下慢慢浮起青色的纹路，歪歪扭扭的几何图，既像电路板，又像老道士画的符。

然后字就直接出现在脑子里，不是耳朵听到的，更像是你本来就知道了：

> 界印绑定完成
> 编号：纪若寒0073
> 属性：数据解析 S级
> 权限：一级观测者

纪若寒干了三年数据分析师，见过乱七八糟上亿条日志，但从来没见过这么玩的——直接把系统装脑子里。

远处一声惨叫。

旁边那辆SUV，司机滚出来，抱着头在地上扭，喊着"进去了！它进去了！"。几分钟后就不动了，再站起来，眼睛全白，四肢着地，跟狗似的闻了闻，嗷一声扑向旁边的大巴。

界兽。

这俩字自己蹦出来，还带解释：被数据吞了自我的活物，靠吸低权限者的界印能量活。

大巴里尖叫起来。

纪若寒靠在车门上，左手腕的界印有点发热，他能"感觉"到那只界兽的能量——一团乱麻，频率不稳，核心比他弱两个档次。

他能看懂。

这就是S级数据亲和？

碎玻璃里飞出个女人，哭着跑出来，界兽在后面追。纪若寒深吸一口气，从后备箱摸出扳手——常年跑野外，车里总得放这么个东西。

他盯着那怪物，能看见能量走的路子，弱点在左脖子，那块最乱，界印还没长牢。

脑子里又冒出来一句：发现弱点，要不要算攻击路径？

纪若寒说要。

马上就有一条线清清楚楚摆在那儿：三步，侧身，砸左颈。

界兽扑过来，纪若寒动了。

三步，侧身，扳手结结实实砸下去。

咔嚓一声，那怪物僵住，回头看他，好像不敢信，然后直挺挺倒下去，身体慢慢变透明，碎成光点，被风吹没了。

又冒字：赢了，拿了17点能量，经验涨了5%。

纪若寒握着扳手喘气，界印凉下来。他抬头往南城看，那地方就在几十公里外，天裂的更大，淡蓝色光把半边天都染了。

苏晴还在那儿。她最后说，星星在流动。

他把扳手塞回去，关上车门，打火继续走。导航没用了，但这条路跑了一百多回，闭着眼都能开。

前面路裂了，一块路面翘起来飘在空中，纪若寒打了把方向，绕过去。

车载电台突然滋啦一声响，然后出来个模糊的女声，像苏晴：

"...若寒...别来...这儿...全是假的..."

然后又没声了。

纪若寒握着方向盘，指节攥得发白。

天变了，规矩重写了，我们全成了试验品。那又怎么样？

我就是干这个的，从乱码里找真相。你在哪儿，我找到哪儿。

前面天缝越来越大，蓝光把整个世界都裹了。

蓝界的天，亮了。
