# 第1章 · 数据异常

高速路的指示牌在往天上飘。

纪若寒握着方向盘，第三次揉了揉眼睛。指示牌还飘在半空中，银灰色的铁皮被看不见的力场揉成一团，像个不正常的云。

车载导航早就瞎了。屏幕上是一片雪花，偶尔蹦出几个乱码：
```
[数 据 流 动 中...]
[编 码 重 构...]
```

他看了一眼副驾的手机。屏保是和苏晴在海边拍的，她笑的很灿烂。最后一条短信停留在十分钟前：

> 哥，南城的空气不对劲，所有电子设备都在乱跳，屏幕上全是奇怪的几何花纹...好像...星星在流动。

纪若寒回她：赶紧去研究所找我，我马上回来。

短信发出，红色感叹号。

然后，通讯断了。

整个世界的通讯，像被一把剪刀剪断了网线。

纪若寒松开方向盘，挂了空挡。车停在应急车道，他推开车门下来，风里带着一股铁锈味，还有点像电路板短路烧糊的味道。

抬头看天。

原本蓝色的天空，裂了一道缝。

缝隙里漏出淡蓝色的光，像一块被烧化的蓝宝石，光里有什么东西在流动——仔细看，那流动的东西竟然是文字：

```
【灵界融合协议 · 启动】
【第99号试炼场 · 蓝星 · 状态：初始化中】
【锚定坐标：73.4°E, 45.2°N...】
```

文字在天空上流动，像LED屏坏了，闪了几下，又变了：

```
【空间叠合进度：17%】
【界印编码中...】
【请保持生命体静止，等待分配...】

纪若寒的后颈忽然一凉。

像是有什么东西从皮肤钻了进去，顺着血管往上走，停在他左手腕内侧。他撸起袖子，看到皮肤下慢慢浮现出一行青色的纹路，复杂的几何图形，像电路板，又像某种古代符文。

【界印绑定成功】
【个体编号：Ji_Ruo_Han/0073】
【亲和属性：数据解析 · S级】
【初始权限：一级观测者】

文字直接出现在他脑子里，不是声音，更像是数据直接写入意识。

纪若寒当过三年数据分析师，见过最复杂的用户行为日志，但从来没见过这么玩的——直接把系统装到人脑子里。

远处传来一声惨叫。

旁边一辆SUV里，司机推开门滚出来，浑身皮肤在发光，他抱着头在地上打滚，嘴里喊着"进去了！它进去了！"——几分钟后，他停止了挣扎，站起来的时候眼睛变成了乳白色，四肢着地，像野兽一样闻了闻空气，然后嗷的一声扑向了旁边的大巴车。

界兽。

这个词莫名出现在纪若寒脑子里，系统自带的词条解释：「被数据侵蚀失去自我的生命体，以吸收低权限者的界印能量为生」。

大巴车里传来尖叫。

纪若寒靠在车门上，左手腕的界印在发热，他能"感觉"到周围的数据流动——那个变成界兽的家伙，能量波动像一团乱麻，频率很不稳定，但核心频率比他低两个档位。

他能看懂。

这就是S级数据亲和？

大巴车的玻璃碎了，一个女人哭着跑出来，界兽在后面追。纪若深吸一口气，从后备箱摸出了一把扳手——那是他常备的修车工具。

他盯着界兽的身体，能看到它能量流动的轨迹，弱点在左颈，那里的数据波动最乱，界印还没长稳。

「检测到可观测弱点，是否编译攻击路径？」

脑子里又弹出提示。

纪若寒在心里说了一个"是"。

立刻，一条清晰的路径出现在意识里——三步，侧身，扳手砸左颈。

界兽扑过来的时候，纪若寒动了。

三步，侧身，精准的一扳手砸下去。

一声脆响，界兽的动作僵住了，它回头，不敢相信的看着纪若寒，然后轰然倒地，身体慢慢透明化，散成了数据碎片，被风吹走了。

【击败F级界兽 · 获得能量 × 17】
【权限经验 + 5%】

纪若寒握着扳手，喘着气，左手腕的界印凉了下来。

他抬头看向南城的方向。

南城就在几十公里外，那里天空的裂缝更大，淡蓝色的光把半边天都染蓝了。

苏晴还在那里。

她发的最后一条短信说：星星在流动。

纪若寒把扳手塞回后备箱，关上车门，重新发动汽车。导航废了没关系，他认识这条路。

不管你变成了什么样，我都来找你。

车开上了通往南城的高速路，前方的路面裂开了，一块浮了起来，悬浮在半空中。纪若寒打了一把方向，绕了过去。

车载电台突然发出了滋滋的电流声，然后一个模糊的女声传了出来，很像苏晴：

"...若寒...别过来...这里...都是假的..."

电流断了。

纪若寒握紧了方向盘，指节发白。

世界已经变了。规则重写了。所有人都被当成了实验品。

但那又怎么样？

他是数据分析师，最擅长的就是从一堆乱码里找出真相。

不管你在哪里，不管你变成了什么样，我都会找到你。

前方，天空的裂缝越来越大，淡蓝色的光笼罩了整个世界。

蓝界黎明，已经开始了。
