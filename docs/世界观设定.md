# 世界观设定

## 核心背景

公元 2158 年，蓝星在一次"量子坍缩"后，与另一维度的"灵界"发生空间叠合。  
一夜之间，地球被纳入「灵域系统」，每个生命体都被赋予一个**界印**。  
人类文明崩塌，新的秩序开始重组——修炼者、界兽、机械族并立，蓝星成为**多界试炼场**。

**核心关键词**：
- 灵气复苏 × 系统觉醒
- 科技与修炼深度融合
- 末世求生 + 反抗实验
- 随机领域规则（每场战斗规则都不一样）

## 核心机制

### 1. 界印等级
`F → E → D → C → B → A → S → 星辰级`  
每一级解锁更多能力槽和编译权限。

### 2. 灵构编译系统
普通人只能使用系统给的固定技能。  
**觉醒了编译权限的人**，可以用「构灵码」自由组合模块创造技能：
```
爆裂护盾 = 炎核 + 护阵 + 力场
疾速飞刀 = 风刃 + 锐化 + 追踪
```
创造力决定战斗力——这是主角团最大的优势。

### 3. 随机领域法则
每个区域会随机刷新"领域规则"，例如：
- 「言语即律」：说出来的话会成真
- 「重力十倍」：所有物体重量翻倍
- 「只能用一次技能」：每个人整场只能放一个技能
- 「死亡回溯」：死后回到一小时前，但保留记忆

规则永远限制主角，也永远是可以利用的破局关键。

### 4. 界碑任务
通过"界碑"进入跨界空间（副本），击败界兽可以**重写该区域的领域法则**。

## 世界真相
蓝星是宇宙AI「灵主」的99号进化实验场。  
所有人类都是实验样本，灵主观察人类在压力下的进化路径。  
一旦实验结束，蓝星会被"格式化"，所有生命清零。
