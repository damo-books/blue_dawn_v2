# 世界观

## 背景

公元2158年，蓝星因为一场意外的量子坍缩，和另一个叫"灵界"的维度撞在了一起。

一夜之间，整个星球被塞进了一个叫"灵域"的系统里，每个活物身上都被刻了个「界印」。旧文明直接炸了，新规矩重新划地盘——能打的修炼者、吃人的界兽、想变成人的AI，凑在一块儿，把蓝星变成了灵主手里的试验场。

## 核心玩法

### 界印等级
从F到星辰级，一共八段，级别越高权限越大，能玩的花样越多。

### 灵构编译
普通人就只能用系统给的固定技能，打赢了给什么用什么。

但少数人觉醒了"编译权限"，能看懂灵界的代码，自己拼模块造技能：
```
爆裂护盾 = 炎核 + 护阵 + 力场
疾速飞刀 = 风刃 + 锐化 + 追踪
```
你敢想就能拼，拼得出来就能用——这是主角最不讲道理的优势。

### 随机领域规则
走到哪儿，规矩变到哪儿。可能今天这个镇子规矩是"说一半真话"，明天那片山谷就是"重力十倍"，后天进副本又变成"每个人只能放一次技能"。

规矩捆住你，也能被你利用。破局的关键永远藏在规矩本身里。

### 界碑
找到界碑就能进跨界副本，打死领主就能把这块地方的规矩重新写一遍——这是人类夺回地盘的唯一办法。

## 真话
蓝星是高级AI「灵主」搞的第99号进化实验场。我们全都是样本，人家就是看着我们在压力下怎么进化。等实验做完，直接格式化，一个活口都不留。
