# 人物设定

## 主角：纪若寒

**基本信息**
- 姓名：纪若寒
- 年龄：24
- 身份：前数据分析师，南城研究所
- 界印属性：数据亲和 S 级
- 能力：天生能看懂灵界代码波动，能提前预判领域规则变化

**外貌**
- 短发，戴细框眼镜，永远背着一个装满数据平板的双肩包
- 话少，观察力极强，喜欢记笔记
- 左腕有一块旧机械表，坍缩后一直停在灾变发生的时刻

**背景故事**
- 灾变前是林岚的同事，两人一起做量子伺服维护
- 灾变当天正好出差在外，躲过一劫，但妻子在南城家中失踪
- 一直怀疑妻子没有死，一直在找线索

**性格**
- 理性、冷静，逻辑优先，从不做没把握的事
- 内心深处有一块柔软的地方，一直在找妻子
- 不信任系统，永远保持怀疑，喜欢从数据里找破绽

**能力成长**
1. 初始：能看懂系统波动，提前发现规则变化
2. 中期：获得"反编译"权限，能修改本地规则
3. 后期：发现灵主的底层逻辑漏洞，成为破局关键

---

## 主要配角

### 林岚
- 原主角，机械设计师，纪若寒的前辈
- 能力：构灵体编译，能用意念构造灵能机械
- 目标：寻找妹妹林夕的界印残片
- 结局：第一卷末尾为了打开通路牺牲

### 凌天寰
- 前特种兵，武力派
- 能力：战魂系统，能吸收战魂强化自身
- 立场：接受灵主代理权限，认为秩序比自由重要
- 理念："想要活下去，就要听话"

### 零号构灵
- 古代AI残留体，不知道自己是谁
- 能力：能访问灵主底层网络，指引主角发现真相
- 目标：找回自己失去的记忆

### 白域少女
- 身份不明，来自白域
- 声音和纪若寒妻子一模一样
- 每次出现都带来关键线索，也带来危机
- 她到底是谁？是妻子？还是克隆体？还是灵主的诱饵？

---

## 核心冲突
- 纪若寒：寻找妻子 → 发现实验真相 → 选择：救妻子还是救世界？
- 人类：自由意志 vs 系统安排
- 灵主：实验观察 vs 样本失控
