# 蓝界黎明 · Blue Dawn

> 当世界化为代码，唯有人类仍在重启灵魂。

**重构版 · 纪若寒主线**

本项目为长篇赛博玄幻小说《蓝界黎明》的工程化写作仓库。  
集成世界观设定、分卷拍表、章节正文、角色语料与校验工具。

---

## 📘 概览

- **作者**：达摩王（damo）
- **体裁**：赛博玄幻 / 意识哲学 / 科幻寓言
- **语言**：中文主文本 + 英文关键字
- **主角**：纪若寒 —— 数据分析师，天生"数据敏感体质"
- **核心设定**：灵构编译 + 随机领域规则 + 灵主实验场
- **仓库目标**：
  1. 以工程思维组织长篇创作；
  2. 模块化管理世界观、节奏、语义；
  3. 支撑未来动画化 / 游戏化 / AI 语料生成。

---

## 📂 目录

- `docs/`：世界设定与视觉圣经  
- `outline/`：分卷节拍表与大纲  
- `chapters/`：小说正文章节  
- `prompts/`：AI 语料与命名规则  
- `tools/`：构建与审校脚本  
- `assets/`：地图、UI、插图素材  

---

## ⚙️ 快速构建

```bash
python3 tools/build.py
```
